- 특정 게시글은 사용자에게 영구 제공되지 않음 (특정 기간 유효성을 지니며 이후 열람 불가) (ex. instagram story)
- 스낵 같은 커뮤니케이션 성격 추구
- 고정 아이디 / 익명성 보장 아이디 두 가지 방식으로 운영
